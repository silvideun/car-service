# Porsche Service Center

Professional car service center landing page build with React, TailwindCSS, and Framer Motion.

## Features
- SEO/GEO Optimized for local search
- Schema.org (JSON-LD) structured data included 
- Responsive modern UI/UX design components
- Integrated Yandex Maps iframe

## Local Development

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Run the development server:
   `npm run dev`

## Deployment

To deploy to GitHub Pages:
1. Run `npm run build`.
2. Deploy the generated `dist` folder to your `gh-pages` branch.
